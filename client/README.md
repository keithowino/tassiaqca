> **Important note:** Dynamic meta tags work in the browser, but Vite's static index.html won't include them at build time . For a single-page app like yours, search engines can still read the tags after JavaScript loads, but for better SEO, consider future server-side rendering or static generation.

### On-Page SEO (The Content)

- Use location keywords naturally in your site content:
- Page titles: "Business Directory for Tassia Complex, Embakasi East"
- Search bar placeholder: "Search for businesses in Tassia, Fedha, or Nyayo..."
- Business descriptions should include local terms like "near Tassia Mall"
- Create location-specific pages (future feature):
    - /tassia-phase-1
    - /tassia-phase-2
    - /fedha-estate
    - /nyayo-estate
- Each page would list businesses in that specific area, with unique descriptions and local keywords
